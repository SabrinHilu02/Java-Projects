# Deduplicator, Banner
## Overview

The assignment consists of two tasks:

- **Deduplikator.java**  
  Reads numbers from the command line, converts them to `double`, and prints each distinct numeric value only once.

- **Banner.java**  
  Prints an integer from the command line as large digits made of `#` characters, including its sign.

This assignment focuses on:

- command-line arguments
- numeric conversion
- duplicate detection
- arrays and loops
- string and character handling
- formatted console output
