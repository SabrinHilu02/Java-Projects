# Recursion

 This project contains my solutions for a university assignment on **recursion** in Java.
## Overview
The assignment consists of two tasks:

- **Roman**: converts integers between 1 and 5000 into Roman numerals
- **Riddle**:   Solves the number puzzle where each number from `1` to `N` must appear exactly twice, with exactly `n` numbers between the two occurrences of `n`.
