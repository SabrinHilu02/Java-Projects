# Weihnachtsplätzchen & Koch-Schneeflocke

## Overview
The assignment consists of two tasks:

- **PlaetzchenPruefer.java**  
  Computes recursively how many Christmas cookies are tested according to a given sampling strategy.

- **KochSchneeflocke.java**  
  Draws a Koch snowflake using a recursive implementation of the Koch curve.


  This assignment focuses on:

- recursion
- divide-and-conquer thinking
- geometric construction
- coordinate-based drawing
- command-line arguments
- recursive graphics
