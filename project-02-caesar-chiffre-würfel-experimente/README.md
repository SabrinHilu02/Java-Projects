# Caesar-Chiffre und Würfel Experimente

## Overview 
The assignment consists of two tasks:
- **Caesar.java**  
  Implements a Caesar cipher for encrypting and decrypting text messages by shifting letters in the alphabet.
- **Wuerfel.java**  
  Simulates dice experiments to estimate how many throws are needed on average until the highest value appears `K` times in a row.

This assignment focuses on:

- strings and character processing
- loops
- command-line arguments
- simulation with random numbers
- formatted console output
